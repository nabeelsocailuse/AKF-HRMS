Projects

Overview

Welcome to the Project Management module of XpertERP, your customized solution for Alkhidmat Foundation Pakistan. This module is designed to help AKFP efficiently plan, execute, and monitor all your projects, from humanitarian aid distributions to community development initiatives. By centralizing project data, tracking progress, managing tasks, and aligning resources, XpertERP ensures that your projects are delivered effectively, transparently, and on schedule. This guide will cover the most commonly used features to help project managers and team members effectively utilize the system, complemented by short training videos for a visual demonstration.

Accessing the Project Management Module

To access the Project Management module in XpertERP:

  1. Log In: Open your web browser and go to the XpertERP login page (e.g., https://your-akfp-xperterp-url.com).
  2. Enter Credentials: Input your assigned Username and Password.
  3. Dashboard: After logging in, you'll see your XpertERP Dashboard.
  4. Navigate to Projects:
    • Click on the "Modules" icon (usually a grid or squares) in the top-left corner.
    • Select "Project" or "Projects" from the list of modules.
    • Alternatively, use the Search Bar at the top by typing "Project" and selecting "Project."

Key Features & Common Tasks

The XpertERP Project Management module supports various functionalities for project managers and team members. Here are some key areas you'll interact with:

Creating a New Project

Project Managers initiate new projects in the system to begin planning and tracking.

  • Process:
    1. Navigate to Project > Project.
    2. Click "+ Add Project".
    3. Enter the "Project Name" (e.g., "Flood Relief Operation - Sindh 2025").
    4. Select the "Project Template" (if available, for standard project types).
    5. Set the "Start Date" and "Expected End Date."
    6. Assign the "Project Manager" and any other relevant "Project Team" members.
    7. Add a brief "Description" of the project's objectives.
    8. Click "Save" then "Submit".

Video: Short video demonstrating how to add a new project, filling in the essential details like name, dates, and manager.

Managing Project Tasks

Tasks are the building blocks of your project. They define the work that needs to be done.

  • Adding a New Task:
    1. Open the relevant "Project" record (from Project > Project).
    2. Scroll down to the "Tasks" section.
    3. Click "+ Add Row" or "Add Task."
    4. Enter the "Task Name" (e.g., "Assess affected areas," "Procure relief supplies").
    5. Assign the "Assigned To" individual or team.
    6. Set the "Expected Start Date" and "Expected End Date."
    7. Set the "Status" (e.g., "Open," "Working," "Completed").
    8. Click "Save" the project document.
  • Updating Task Status and Progress:
    1. Open the relevant "Project" record or navigate directly to Project > Task.
    2. Open the specific "Task."
    3. Update the "Status" (e.g., from "Open" to "Working" to "Completed").
    4. Update "Progress (%)" to reflect how much of the task is done.
    5. Add "Comments" or "Notes" on progress or blockers.
    6. Click "Save" then "Submit" (if changes need to be reflected in workflow).

Tracking Project Progress

Monitor the overall health and status of your projects.

  • Viewing Project Dashboard:
    1. Navigate to Project > Project.
    2. The list view provides an overview of all projects, including their "Status" and "Progress (%)."
    3. Click on a project to view its detailed dashboard, which often includes a summary of tasks, timelines, and more.
  • Gantt Chart (Optional, if configured):
    • Some projects may have a Gantt Chart view available within the project record, offering a visual representation of tasks and their dependencies over time.

Linking Resources (Optional)

The Project module can be integrated with other modules to link resources.

  • Linking Stock/Inventory: (If integrated) You can link "Material Requests" or "Stock Entries" directly to a project to track consumption of relief supplies or materials.
  • Linking Financials: (If integrated) Project expenses or budgets from the Accounting module can be associated with projects for financial tracking.

Reports

The Project Management module offers various reports to help analyze project performance and resource utilization.

  • Accessing Reports:
    1. Navigate to Project > Reports.
    2. Select the desired report from the list (e.g., "Project Progress Report," "Task Status Report," "Timesheet Summary by Project").
    3. Apply relevant filters (e.g., by Project Manager, Project Status, Date Range) and click "Refresh" to generate the report.
